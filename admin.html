หมายเหตุ / วิธีใช้
เพิ่มวิดีโอ: เพิ่มแถวใน Sheet วิดีโอ → refresh เว็บ จะเห็นทันที
เพิ่ม diary entry: เพิ่มแถวใน Sheet ไดอารี่ → refresh เว็บ ใส่รูปใน images/ บน GitHub
เพิ่มหัวข้อ gallery: แก้ไฟล์ index.html ในส่วน const TOPICS แล้วอัปโหลดขึ้น GitHub
เปลี่ยน vote topics: แก้ไฟล์ index.html ในส่วน const VOTE_TOPICS → vote เก่าจะ reset อัตโนมัติ
