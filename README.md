# ห้องสมุดเพื่อนโยคะ
### by Yoga with Jib

---

**แหล่งรวมข้อมูลออนไลน์เพื่อคนที่รักการฝึกโยคะและการสอนโยคะ**

เราเชื่อว่าความรู้และประสบการณ์ที่แต่ละคนสะสมมานั้นมีคุณค่ามาก และควรได้รับการแบ่งปัน ไม่ว่าจะเป็นเรื่องการฝึก การสอน การเงิน การจัดการสตูดิโอ หรือชีวิตของครูโยคะในแบบที่ไม่มีใครพูดถึง

---

## 🌿 คลังนี้คืออะไร

เป็นพื้นที่ออนไลน์ที่เราช่วยกันสร้างขึ้นมา เนื้อหาทุกชิ้นมาจากการพูดคุย แลกเปลี่ยน และถอดบทเรียนร่วมกันของคอมมูนิตี้ครูโยคะ ไม่ใช่ตำราที่เขียนโดยผู้เชี่ยวชาญคนเดียว แต่คือประสบการณ์จริงจากหลายๆ คนที่เดินเส้นทางเดียวกัน

หัวข้อที่เราพูดคุยกัน:

- **ทักษะการสอน** — เทคนิค cueing งานวิจัยใหม่ กรณีศึกษาจากชั้นเรียน การดัดแปลงท่า
- **ธุรกิจและการเงิน** — ภาษี การตั้งราคา การออกใบเสร็จ การวางแผนเกษียณในฐานะฟรีแลนซ์
- **สตูดิโอ** — ซอฟต์แวร์จัดการคลาส อุปกรณ์ การตลาดออนไลน์
- **ชีวิตครูโยคะ** — burnout การวางขอบเขต เส้นทางอาชีพ self-practice
- **วิดีโอ** — คลิปจาก Yoga with Jib ทั้งฟรีและสำหรับสมาชิก
